# Flowers Recognition
Classification Flowers into 5 category (Daisy, Dandelion, Rose, Sunflower, Tulip)

## Model Structure
- 8 Unit Convolutional 2D
- MaxPooling2D
- 16 Unit Convolutional 2D
- MaxPooling2D
- Flatten
- 128 Unit Dense
- Dropout: 0.2
- 256 Unit Dense
- 5 Unit Dense

## Model Accuracy (for 10 epochs)
- Training accuracy: 64.11 %
- Validation accuracy: 68.44 %
